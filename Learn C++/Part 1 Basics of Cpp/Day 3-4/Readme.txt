1. Biến và kiểu dữ liệu:
   Biến là một phần tử lưu trữ dữ liệu trong bộ nhớ máy tính. Mỗi biến có một tên duy nhất và một kiểu dữ liệu xác định. Các kiểu dữ liệu cơ bản bao gồm:
     - Số nguyên (int)**: Lưu trữ các số nguyên như -5, 0, 25. 
     - Số thực (float)**: Lưu trữ các số thực như 3.14, -2.5, 7.0.
     - Ký tự (char)**: Lưu trữ một ký tự đơn như 'a', 'x', '9'.
     - Chuỗi (string)**: Lưu trữ các chuỗi ký tự như "Xin chào", "Lập trình cơ bản".
     - Boolean (bool)**: Lưu trữ các giá trị logic true hoặc false.

2. Toán tử số học và gán:
     Các toán tử số học cơ bản bao gồm:
     - Cộng (+), Trừ (-), Nhân (*), Chia (/), Chia lấy phần dư (%).
     Ví dụ: a = 10; b = 5; c = a + b; // c = 15.
     Các toán tử gán bao gồm =, +=, -=, *=, /=, `%=.
     Ví dụ: a += 3; // a = a + 3 = 13.
