
1. **Cách tạo và sử dụng custom modules**:
   - Modules trong Python là các tệp tin (`.py`) chứa các hàm, lớp và biến. Chúng cho phép bạn tái sử dụng mã nguồn và tổ chức code của mình một cách hiệu quả.
   - Để tạo một custom module, bạn chỉ cần tạo một tệp tin `.py` và định nghĩa các hàm, lớp hoặc biến bên trong nó.
   - Để sử dụng một custom module, bạn cần `import` nó vào chương trình của mình bằng lệnh `import` hoặc `from ... import ...`.
   - Bạn có thể sử dụng các chức năng, lớp hoặc biến trong module bằng cách gọi chúng với cú pháp `module_name.function_name()`, `module_name.class_name()` hoặc `module_name.variable_name`.

2. **Tái sử dụng mã nguồn qua modules**:
   - Modules cho phép bạn tái sử dụng mã nguồn một cách hiệu quả. Bằng cách tách các chức năng, lớp hoặc biến vào các modules riêng biệt, bạn có thể gọi chúng ở bất cứ đâu trong chương trình của mình.
   - Điều này giúp code của bạn trở nên dễ đọc, dễ bảo trì và dễ mở rộng hơn.
   - Bạn cũng có thể chia sẻ các modules của mình với người khác hoặc sử dụng các modules được phát triển bởi cộng đồng Python.
