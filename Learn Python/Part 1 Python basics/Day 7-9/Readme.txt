- **Các kiểu dữ liệu**:
  - **List**: Danh sách các phần tử, thứ tự, có thể chứa các kiểu dữ liệu khác nhau. Khai báo: `my_list = [1, 2.5, "hello"]`.
  - **Tuple**: Giống list nhưng các phần tử bất biến. Khai báo: `my_tuple = (1, 2.5, "hello")`.
  - **Set**: Tập hợp các phần tử duy nhất, không có thứ tự. Khai báo: `my_set = {1, 2.5, "hello"}`.
  - **Dictionary**: Tập hợp các cặp key-value. Khai báo: `my_dict = {"name": "John", "age": 30}`.

- **Các cấu trúc điều kiện**:
  - **if-elif-else**: Kiểm tra và thực hiện các khối lệnh tương ứng với điều kiện.
  - **Các phép so sánh**: `==` (bằng), `!=` (khác), `>` (lớn hơn), `<` (nhỏ hơn), `>=` (lớn hơn hoặc bằng), `<=` (nhỏ hơn hoặc bằng).
